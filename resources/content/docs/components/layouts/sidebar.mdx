## Intent
The sidebar comes in three variations: `"sidebar"`, `"floating"`, and `"inset"`. By default, the variation is set to `"sidebar"`.

## Collapsible
The sidebar can be made collapsible by using the `collapsible` prop in the `SidebarProvider`. The default collapsible behavior is `"hidden"`.

## Props
The sidebar supports the following props:
### Sidebar
| **Property**  | **Type**                             | **Default** |
|---------------|--------------------------------------|-------------|
| `intent`      | `"default" \| "floating" \| "inset"` | `"default"` |
| `collapsible` | `"hidden" \| "dock" \| "none"`       | `"hidden"`  |
| `side`        | `"left" \| "right"`                  | `"left"`    |

### SidebarContextProps

| **Property**        | **Type**                    |
|---------------------|-----------------------------|
| `state`             | `"expanded" \| "collapsed"` |
| `open`              | `boolean`                   |
| `setOpen`           | `(open: boolean) => void`   |
| `isOpenOnMobile`    | `boolean`                   |
| `setIsOpenOnMobile` | `(open: boolean) => void`   |
| `isMobile`          | `boolean`                   |
| `toggleSidebar`     | `() => void`                |

### SidebarProviderProps

| **Property**   | **Type**                  | **Default** |
|----------------|---------------------------|-------------|
| `defaultOpen`  | `boolean`                 | `true`      |
| `isOpen`       | `boolean`                 | `undefined` |
| `onOpenChange` | `(open: boolean) => void` | `undefined` |
