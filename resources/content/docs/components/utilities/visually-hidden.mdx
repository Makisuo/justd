## Visually Hidden

Visually Hidden is a component that hides an element from the screen but still keeps it accessible to screen readers.

```tsx
import { VisuallyHidden } from 'ui'

function App() {
  return (
    <div>
      <VisuallyHidden>This is a hidden element</VisuallyHidden>
    </div>
  )
}
```
