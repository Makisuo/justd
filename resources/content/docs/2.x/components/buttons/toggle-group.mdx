## Orientation
By default, the toggle button group is horizontally oriented. You can change it to a vertical orientation by using the `orientation` prop.
<How toUse="buttons/toggle-group/toggle-group-orientation-demo" />

## Selection Mode
You can change the selection mode of the toggle button group by using the `selectionMode` prop.
<How toUse="buttons/toggle-group/toggle-group-selection-mode-demo" />

## Appearance
You can change the appearance of the toggle button group by using the `appearance` prop.
<How toUse="buttons/toggle-group/toggle-group-appearance-demo" />

## Gap
By default, the toggle button group has no gap between the buttons. You can change the gap by using the `gap` prop.
<How toUse="buttons/toggle-group/toggle-group-gap-demo" />
<Note intent='warning'>
    Never ever set the gap to `0` when you want to use the `plain` appearance.
</Note>

## Size
By default, the toggle button group has a medium size. You can change the size by using the `size` prop.
<How toUse="buttons/toggle-group/toggle-group-sizing-demo" />

## Controlled
You can control the state of the toggle button group by using the `selectedKeys` prop.
<How toUse="buttons/toggle-group/toggle-group-controlled-demo" />

## Disabled
You can disable an entire toggle group by setting the `isDisabled` prop on the `ToggleGroup` component, which disables all toggles in the group. Alternatively, you can disable specific toggle buttons individually by setting the `isDisabled` prop on the `Toggle` component.
<How toUse="buttons/toggle-group/toggle-group-disabled-demo" />
